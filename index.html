Custom-Editable-Combobox

A jQuery plugin that creates custom editable select lists.

Custom-Editable-Combobox allows you to quickly create custom editable combobox for use in web applications and websites. The combobox is easily customisable changing the css styles and provides flexability with options in select.

Features

  • Straightforward implementation with simple options
  • Can run as many editable comboboxes as required
  • Comboboxes can be attached to any element required
  • Comboboxes look and feel is easily customised through the small css file

Usage

Include the plugin javascript file along with jquery <script src="jquery.min.js"></script> <script src="custom-combobox.js"></script>

Include css file <link href="custom-combobox.css" rel="stylesheet" />

Define your combobox HTML <select id="customCombobox"> <option value="val 1" selected="selected">val 1</option> <option value="val 2">val 2</option> <option value="val 3">val 3</option> <option value="val 4">val 4</option> </select>

Attach the toolbar to an element passing in your options as an object. Available options are detailed below. var combobox = $('#customCombobox').customEditableCombobox(options);

Options

option - description *is_editable - Choose if you want the combobox to be editable. Default: true.
