Custom-Editable-Combobox

A jQuery plugin that creates custom editable select lists.

Custom-Editable-Combobox allows you to quickly create custom editable combobox for use in web applications and websites. The combobox is easily customizable by changing the css styles and provides flexibility with options in select.

Features

  • Straightforward implementation with simple options
  • Can run as many editable comboboxes as required
  • Comboboxes can be attached to any element required
  • Comboboxes look and feel is easily customized through the small css file

Examples

Custom editable combobox

HTML

<select id="customEditableCombobox">
        <option value="val 1" selected="selected">val 1</option>
        <option value="val 2">val 2</option>
        <option value="val 3">val 3</option>
        <option value="val 4">val 4</option>
    </select>

Javascript

var cmb = $('#customEditableCombobox').customEditableCombobox();

Custom non-editable combobox

HTML

<select id="customCombobox">
        <option value="val 1" selected="selected">val 1</option>
        <option value="val 2">val 2</option>
        <option value="val 3">val 3</option>
        <option value="val 4">val 4</option>
    </select>

Javascript

var cmb = $('#customCombobox').customEditableCombobox({
                is_editable: false
            });

Usage

Include the plugin javascript file along with jquery

<script src="jquery.min.js"></script>
<script src="custom-combobox.js"></script>

Include css file

<link href="custom-combobox.css" rel="stylesheet" />

Define your combobox HTML

<select id="customCombobox">
	<option value="val 1" selected="selected">val 1</option>
	<option value="val 2">val 2</option>
	<option value="val 3">val 3</option>
	<option value="val 4">val 4</option>
</select>

Attach the combobox to an element passing in your options as an object. Available options are detailed below.

var combobox = $('#customCombobox').customEditableCombobox(options);

Options

OptionDescription
is_editable Choose if you want the combobox to be editable. Default: true.

Methods

Below is a list of methods available on the elements that already have a editable combobox instantiated

MethodArgumentsDescription
getSelectedValue None Returns currently selected value
add Value, Text, Position Add new option to combobox
remove Position Removes option from chosen position
selectItem Position Select option at chosen position
unload None Unloads the custom combobox element

Events

Below are a list of events that are triggered when certain events happen with the editable combobox. You can listen for these events using the .on method.

For example. 

$('#element').on('selectedOptionChanged',
    function( event ) {
        // this: the element the custom combobox is attached to
    }
);

EventDescription
optionsShowing Triggered before the options are shown.
optionsShowed Triggered after the options are shown.
optionsClosing Triggered before the options are closed.
optionsClosed Triggered after the options are closed.
inputFocused Triggered when the combobox input get focus.
selectedOptionChanged Triggered when the new item is selected.
